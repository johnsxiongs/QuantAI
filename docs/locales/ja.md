# QuantAI 日本語

QuantAI は、株式、暗号資産、市場テーマ、ウォッチリスト、投資仮説を追う人のための AI 投資リサーチ助手です。

市場の問いに文脈が必要なときに使えます。何が変わったのか、どのシグナルが重要か、どの前提を確認すべきか、投資仮説をどう更新すべきかを整理します。

## 調べられること

- 株式、決算、開示資料、バリュエーション、ウォッチリストの変化
- 暗号資産、ビットコイン、市場ナラティブ、リスク要因
- テクニカル分析、ファンダメンタル分析、定量シグナル、ファクターアイデア
- 新しい証拠に応じて見直す投資仮説
