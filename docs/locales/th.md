# QuantAI ภาษาไทย

QuantAI คือผู้ช่วยวิจัยการลงทุนด้วย AI สำหรับผู้ที่ติดตามหุ้น สินทรัพย์คริปโต ธีมตลาด watchlist และสมมติฐานการลงทุน

ใช้ QuantAI เมื่อคำถามด้านตลาดต้องการบริบท: อะไรเปลี่ยนไป สัญญาณใดสำคัญ สมมติฐานใดควรตรวจสอบ และ thesis ควรถูกปรับอย่างไร

## สิ่งที่คุณสามารถวิจัยได้

- หุ้น ผลประกอบการ เอกสารบริษัท คำถามด้านมูลค่า และการเปลี่ยนแปลงใน watchlist
- สินทรัพย์คริปโต bitcoin เรื่องเล่าของตลาด และปัจจัยเสี่ยง
- การวิเคราะห์ทางเทคนิค การวิเคราะห์ปัจจัยพื้นฐาน สัญญาณเชิงปริมาณ และแนวคิด factor
- สมมติฐานการลงทุนที่ต้องทบทวนเมื่อมีหลักฐานใหม่

## ขอบเขตการใช้งานอย่างรับผิดชอบ

QuantAI ช่วยด้านการวิจัยและการวิเคราะห์ ไม่ใช่นายหน้า ไม่ใช่บอทเทรดอัตโนมัติ ไม่ใช่ระบบส่งคำสั่งซื้อขาย ไม่ใช่บริการสัญญาณที่รับประกันผล ไม่ใช่ผู้จัดการพอร์ต และไม่ใช่ที่ปรึกษาการเงินส่วนบุคคล
