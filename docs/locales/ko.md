# QuantAI 한국어

QuantAI는 주식, 암호화폐 자산, 시장 테마, 관심 목록, 투자 thesis를 따라가는 사람들을 위한 AI 투자 리서치 어시스턴트입니다.

시장 질문에 맥락이 필요할 때 QuantAI를 사용할 수 있습니다. 무엇이 바뀌었는지, 어떤 신호가 중요한지, 어떤 가정을 확인해야 하는지, thesis를 어떻게 업데이트해야 하는지 정리합니다.

## 리서치할 수 있는 것

- 주식, 실적, 공시, 밸류에이션 질문, 관심 목록 변화
- 암호화폐 자산, 비트코인 질문, 시장 내러티브, 위험 요인
- 기술적 분석, 기본적 분석, 정량 신호, 팩터 아이디어
- 새로운 증거가 생길 때 다시 검토해야 하는 투자 thesis

## 책임 있는 사용 범위

QuantAI는 리서치와 분석을 돕습니다. 브로커, 자동 거래 봇, 주문 라우팅 시스템, 보장된 신호 서비스, 포트폴리오 매니저 또는 개인화된 금융 자문사가 아닙니다.
