# QuantAI 繁體中文介紹

QuantAI 是 AI 投資研究助手，適合關注股票、加密資產、市場主題、觀察清單和投資假設的使用者。

當一個市場問題牽涉財報、公告、價格走勢、技術指標、基本面、量化信號和後續追問時，QuantAI 可以幫你把研究脈絡整理得更清楚。

## 你可以研究什麼

- 股票、財報、公告、估值問題和觀察清單變化
- 加密資產、比特幣市場問題、市場敘事和風險因素
- 技術分析、基本面分析、量化信號和因子思路
- 需要持續複盤的投資假設和市場觀點

- 官網：[https://aiquant.io](https://aiquant.io)
- Web App：[https://app.aiquant.io](https://app.aiquant.io)

## 責任邊界

QuantAI 支援研究和分析。它不是券商、自動交易機器人、下單系統、保證信號服務、投資組合管理工具或個人化金融顧問。
